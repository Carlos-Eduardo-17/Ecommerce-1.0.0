## Descripción
Este proyecto es una aplicación backend desarrollada como proyecto final del Curso de Especialización en Desarrollo Backend ofrecido por IDAT. La aplicación está diseñada para gestionar y procesar datos relacionados como por ejemplo: "gestión de usuarios", "procesamiento de pedidos", etc.

A su vez está puesta en producción en Render, lo que permite a los usuarios acceder a la aplicación desde cualquier lugar y en cualquier momento ( https://ecommerce-1-0-0.onrender.com/api/health ).

La aplicación está construida utilizando tecnologías modernas y sigue las mejores prácticas de desarrollo backend para garantizar un rendimiento óptimo, escalabilidad y seguridad.
Mayor información sobre el proyecto, incluyendo la arquitectura, las tecnologías utilizadas y las funcionalidades implementadas, se puede encontrar en la documentación del proyecto (docs).
